# 🐭 ピクセル・ダンジョン：動的迷路 (Pixel Mouse Maze)

[![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-Live-brightgreen)](https://biology-c.github.io/mousemaze/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

ブラウザベースの純粋なJavaScriptで作られた、ドット絵スタイルの迷路探索ゲームです。方向感覚と戦略を駆使して、出口を目指しましょう！

**🏠 ゲームをプレイ：[https://biology-c.github.io/mousemaze/](https://biology-c.github.io/mousemaze/)**

---

## 🌟 ゲームの特徴

*   **18の進行レベル**: 移動、壁掘り、灯台、敵への対処など、基礎を学べる6つのチュートリアルステージを含みます。
*   **称号システム**: チュートリアル完了で「🐾 初心者」から「⚔️ 勇者ネズミ」までの称号を獲得。
*   **🐍 ヘビの敵システム**:
    *   60秒ごとにヘビが出現！捕まるとゲームオーバー。
    *   ヘビは灯台を優先して追いかけ、灯台がない場合はプレイヤーを追いかけます。
    *   20秒ごとに2秒間の猛ダッシュを行います。
    *   ヘビに向き合って `Space` キーで攻撃！3回で退治できます。
*   **🔦 灯台ビーコン**: 霧を照らす永久的な灯台を設置。ただし、道も塞いでしまうため、ヘビを誘導するなどの戦略が必要です。
*   **⚡ スマートアクション**: 正面が壁なら「壁掘り」、壁がなければ「ヘビへの攻撃」を `Space` ひとつで実行。
*   **多彩なアイテム**:
    *   🧀 **エナジーチーズ**: 壁掘りの回数を増やします。
    *   🍄 **魔法のキノコ**: 一時的にマップ全体が見えるようになります。
    *   💎 **原鉱**: 視野の半径を永久的に広げます。
    *   🌀 **テレポート**: ペアになった場所へ一瞬で移動。
*   **3段階の難易度**: ヘブン（壁掘り21回）、ノーマル（15回）、ファミン（9回）。
*   **ランキング**: クリアタイムを記録し、累計プレイ時間に応じた称号を獲得。
*   **詰み防止機能**: 自分の灯台で道を塞いでしまった場合、自動的に検知して再開します。

## 🎮 操作方法

### PC (キーボード)
| キー | 機能 |
|------|------|
| `↑↓←→` / `WASD` | 移動 |
| `Space` | 壁を掘る / ヘビを攻撃 |
| `Q` | 灯台を設置 (目の前) |
| `Z` | ヒント (ルート表示) |
| `Esc` | 一時停止 |
| `imsupergm` | GMモード |

### スマホ / タブレット (タッチ)
*   **左下**: 移動用アナログジョイスティック。
*   **右下**: スキルボタン (アクション、ヒント、灯台、設定)。
