# 🐭 像素地城：動態迷宮 (Pixel Mouse Maze)

[![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-Live-brightgreen)](https://biology-c.github.io/mousemaze/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

一個基於瀏覽器、純 JavaScript 開發的像素風迷宮探索遊戲。挑戰您的方向感與反應，利用道具與智慧在迷宮中尋找出口！

**🏠 遊戲網址：[https://biology-c.github.io/mousemaze/](https://biology-c.github.io/mousemaze/)**

---

## 🌟 遊戲特色

*   **18 關漸進式挑戰**：包含 6 關教學關卡，從移動、鑽牆、燈塔、道具到敵人應對，循序漸進學習所有技巧。
*   **教學稱號系統**：完成教學關卡獲得獨特稱號（🐾 初學者 → ⚔️ 勇者鼠）。
*   **🐍 蛇敵人系統**：
    *   每 60 秒隨機生成一隻蛇，被碰到就失敗！
    *   蛇會追蹤燈塔，沒有燈塔時追玩家。
    *   每 20 秒加速衝刺 2 秒。
    *   面向蛇身按空白鍵攻擊，3 次消滅！
*   **🔦 燈塔記號**：放置永久燈塔照亮迷霧，但會變成牆壁。策略性放置可誘導蛇的行進方向！
*   **⚡ 空白鍵智慧動作**：面前有牆→鑽牆（消耗次數）；面前無牆→攻擊（不消耗）。
*   **多樣化道具**：
    *   🧀 **能量起司**：增加鑽牆次數。
    *   🍄 **魔法蘑菇**：短暫獲得全地圖視野。
    *   💎 **原礦**：永久提升角色的視野範圍。
    *   🌀 **傳送陣**：雙向遠距離傳送。
*   **三種難度**：天堂（鑽牆×21）、一般（×15）、飢荒（×9）。
*   **排行榜**：關卡紀錄 + 遊玩時間排行（含累計時間稱號）。
*   **困死偵測**：把自己困住時自動提示並重來。
*   **GM 金手指**：隱藏 GM 模式，可監視數值並切換迷霧。
*   **RWD 自適應排版**：全面支援手機瀏覽，內建類比搖桿操作。

## 🎮 控制說明

### 電腦版 (鍵盤)
| 按鍵 | 功能 |
|------|------|
| `↑↓←→` / `WASD` | 移動 |
| `Space` | 面前有牆→鑽牆；無牆→攻擊蛇 |
| `Q` | 放置燈塔（面前方向） |
| `Z` | 路線提示 |
| `Esc` | 暫停 |
| `imsupergm` | GM 模式 |

### 手機/平板 (觸控)
*   **左下角**：類比搖桿區控制移動。
*   **右下角**：技能按鈕區（動作、提示、燈塔、設定）。

## 🛠️ 技術開發

*   **語言**：HTML5, CSS3, Vanilla JavaScript（純 JS, 無依賴項目）
*   **渲染**：Canvas API（多光源迷霧系統）
*   **核心演算法**：
    *   **迷宮生成**：隨機 Prim 演算法
    *   **路徑搜尋**：BFS（廣度優先搜尋）用於提示導航與困死偵測
    *   **蛇 AI**：貪婪尋路（追燈塔/追玩家）+ 速度週期變化
*   **資料儲存**：LocalStorage（進度、排行榜、遊玩時間）
